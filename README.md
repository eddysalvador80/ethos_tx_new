# ETHOS·TX — Apoyo a la decisión en transfusión y hemorragia perioperatoria

Módulo educativo y de apoyo a la decisión clínica para anestesiólogos, parte de la suite **ETHOS** (Lumina Health). Cubre:

- Decisión transfusional (umbrales restrictivos, productos)
- Protocolo de transfusión masiva (MTP) y diamante letal
- Timing perioperatorio de anticoagulantes (PAUSE)
- Reversión de anticoagulantes y DOACs
- Bridging y bloqueo neuroaxial
- Antiagregantes perioperatorios (ASRA 2025)
- Coagulopatía obstétrica (WOMAN, SOAP)
- **Factor VIIa recombinante como rescate** *(nuevo en v19)*

## Uso

Abrir `index.html` (o `demos/ethos-tx-v19.html`) en cualquier navegador moderno. No requiere servidor ni conexión a internet.

## Versiones

| Versión | Cambios principales |
| --- | --- |
| **v19** | Añade lección "Factor VIIa recombinante como rescate" en Bloque C (Hemorragia masiva), con tablas de precondiciones, evidencia off-label, riesgo trombótico cuantificado (Levi NEJM 2010, AHRQ Review) y caso clínico interactivo |
| v18 | Decisión transfusional + Timing anticoagulación + Bridging + Safety Gate |

## Aviso

Herramienta educativa de apoyo a la decisión. No reemplaza el juicio clínico ni los protocolos institucionales. El uso off-label de cualquier fármaco mencionado requiere consentimiento informado.
